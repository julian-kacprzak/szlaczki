# szlaczki

Do robienia szlaczkow prowadzacego.

# instalacja

``` r
install.packages("devtools") 
devtools::install_github("julian-kacprzak/szlaczki")
```

# uzycie

``` r
szlaczki(tekst = "Lista 1", wys = 7, szer = 79, symbole = 2, hlinie = 2)
```
